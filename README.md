# challenge
Simple Spring boot application which provide RESTful API for money transfer

### Prerequisite
- Maven
- JDK 1.8+
### Project Structure
```bash
challenge
├── src
│   ├── main
│   │   ├── java\com\dws\challenge
│   │   └── resources
│   └── test
│       ├── java\com\dws\challenge
│       └── resources
├── build.gradle
├── gradlew
├── HELP.md
└── README.md
```
### Packaging
```
gradlew.bat runJar
```
